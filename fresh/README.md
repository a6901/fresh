###登录接口
~~~
URL：user/login
data：{
  telephone:
  passWord:
}
return:
{
    code: 状态码,
    message: "返回消息",
    data: {
	username:"用户名",
    telephone:"用户的电话号码"
}
~~~
code|message|data
:--|:--:|:--:   
100|登录成功|
200|账号或密码错误|

###注册接口

    email:"电子邮箱",
}
return：{
	code:"状态码",
    message:"状态信息",
